# Pixlift Landing Page

A modern landing page project for **Pixlift | Local AI Image Enhancement** built with **Vite**, **React**, **TypeScript**, and **Tailwind CSS**.

## Overview

This repository contains a polished landing page concept for an AI-powered image enhancement product. The interface is designed with a dark, cinematic aesthetic and uses **custom typography**, **GSAP animations**, and **responsive layout styling**.


## Notes

- The current implementation is focused on the landing page experience.
- If you want to extend this into an interactive React application, add content inside `src/App.tsx` and mount it through the existing React entrypoint.
